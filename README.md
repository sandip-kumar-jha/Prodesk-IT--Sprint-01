# Prodesk IT Landing Page

## Overview

This project is a responsive landing page developed for Sprint 01 of the Prodesk IT Trainee Engineer Program.

## Features

* Responsive Navbar
* Hero Section
* Services Section
* Why Choose Us Section
* Footer with Social Icons
* Dark / Light Mode Toggle
* Sticky Navigation Bar
* Hover Effects
* Mobile Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript
